# Nexora AI Product Studio

A polished, responsive landing page for an AI product studio concept. The site is built with Next.js and React, using supplied visual assets, original logo lockups, smooth scroll-triggered motion, sticky navigation, fullscreen-style sections, and clean editorial layout details.

## Features

- Next.js App Router
- React components split by section
- Responsive fullscreen-style landing page
- Sticky professional navigation
- Scroll-triggered staggered animations
- Supplied image assets in `public/images`
- Logo assets in `public/logos`
- Geist typography through `next/font`

## Getting Started

Install dependencies:

```bash
npm install
```

Run the local development server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```

## Image Assets

Place or replace section artwork in:

```text
public/images
```

Logo assets live in:

```text
public/logos
```
